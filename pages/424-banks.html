4.2.4 Banks

4.2.4.1 Banks HomePage

Description

This page allows the user to be able to view the banks which are saved in the database. The user can then proceed to add more banks, search, update or delete the banks.

 

 

Reference:

Control:

Functionality Description:

1

Bank link *located at the dashboard

Click here to view the bank homepage.

 

2

Add Bank button

Click here to add a new bank. The system will redirect you to the Add bank page.

3

Search Bank Input link

Click here to enter in the bank being searched for.

4

Delete button

Click here to delete a bank.

5

Update button

Click here to update banking details. The system will redirect you to the update bank page

6

Show a number of entries

Click here to show only a certain number of entries.

7

Username Link

Click here to sign out of the website

•         Possible Error Messages: No data available in table

4.2.4.2 Add Bank

Description

This page allows the user to be able to add a new bank. The user enters the bank details in the relevant input boxes and the system saves the details in the database.

 

Reference:

Control:

Functionality Description:

1

Name of bank Input Box

Click here to enter the name of the back.

 

2

Branch Code Input Box

Click here to enter the bank branch code.

3

Add button

Click here to save new banking details.

4

Return to list button

Click here to return to the update bank homepage.

•         Possible Error Messages: Name of bank input box → Name of bank is required

      Branch code input box → Branch code is required
