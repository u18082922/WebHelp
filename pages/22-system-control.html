2.2 System Control

Control:

Description:

Menu Item

The menu item is the list of items displayed on the left-hand side of the screen. This allows quick and easy navigation between pages. When selected the name of the specific menu item will be displayed on the screen on the side.

 

 

Input boxes

Input boxes can be used to enter specific required details.

 

Drop down list

Drop down lists contain a list of details stored in the database. They provide more than one option where the user can choose the desired record to enter.

Grid view

A grid view displays data stored in the database, in table format.

Tabs

Details are displayed on the same screen, just in different tabs. It sections the details being displayed into separate tabs.
