4.2.1 Login Functions

 

4.2.1.1 Login page

Description

This page allows the user to login in to the InsureTech website. The user needs to navigate to enter their username and their password. The user will then be redirected to the system dashboard screen.

Reference:

Control:

Functionality Description:

1

Enter Username Textbox

Type the username in the textbox.

 

Tip*

Username is your email address

2

Enter Password Textbox

Type the password in the textbox.

 

 

Note*

Password is case sensitive

3

Forgot Password link

Click here to redirect to the reset password page.

 

4

Not yet Registered link

Click here to register and gain access to the system.

5

Login

Click here to login to the system

 

•         Possible Error Messages:

o   Incorrect username/password
